Generate JSON First

Example commands and export guidance

Use these examples to produce upload-ready scanner outputs. Keep the JSON files, then upload them here to build the consolidated VAPT report.

Trivy filesystem scan

Scan the current project directory

Best for dependency and filesystem findings from the working tree.

trivy fs --format json --output trivy-report.json .
Trivy image scan

Scan a container image

Use this when your security source is a built image instead of a local file tree.

trivy image --format json --output trivy-image-report.json <image-name>:<tag>
Bandit

Generate Python SAST output

Recursively scan the project and save findings in upload-ready JSON format.

bandit -r . -f json -o bandit-report.json
ZAP

Export an alert report as JSON

From the ZAP UI or your existing automation, export the alerts report in JSON format and upload that exported file here. 

ZAP UI -> Report / Export -> JSON
Save the file, then upload it in this dashboard.
