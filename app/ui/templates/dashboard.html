DevSecOps Report Generator

Upload one or more JSON reports from Trivy/Bandit/ZAP and generate grouped output.

How to get JSON files

This uploader accepts JSON reports that match the formats parsed by this app: Trivy reports with Results, Bandit reports with results and metrics, and ZAP reports with site and alerts.

Trivy

Use JSON output and save it to a file before uploading.

Filesystem, image, misconfiguration, and secret results are supported.

Bandit

Run Bandit with -f json and write the result to a file.

The parser reads the standard Bandit JSON structure.

ZAP

Export a JSON report from ZAP, or generate one with the baseline scan helper.

The JSON should include the usual site and alerts sections.

Trivy filesystem 
trivy fs --format json --output trivy-fs-report.json .
Copied
Trivy image 
trivy image --format json --output trivy-image-report.json your-image:tag
Copied
Bandit 
bandit -r . -f json -o bandit-report.json
Copied
ZAP baseline 
zap-baseline.py -t https://example.com -J zap-report.json
Copied
